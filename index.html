Methodology

The feed is a credibility surface, not a scoreboard. It exposes how PropellerSwap reached a route decision using Fynd over Tycho state, and separates what is verifiable today from future optimality claims.

Current simulation boundary

Only the auction events are simulated in this PoC. The Tycho proxy, Fynd build, local server, event ingestion endpoint, filtering, route expansion, and export path are production-shaped surfaces. Once data/events.json contains real solver events, the UI switches from simulation to production event mode automatically.

Why losses are shown

Showing losses makes wins credible. A feed that only surfaces wins is marketing. This feed shows all outcomes — wins, losses, timeouts — so solvers evaluating Fynd + Tycho see the full distribution, not a curated sample.

Benchmark cycle

A production event fires when PropellerSwap receives a CoW auction, computes a Fynd route, submits or skips, and records the auction result. The dashboard renders the event as an evidence trail — not a private internal log.

Route inspection

Expanded rows show every candidate route Fynd evaluated during the solve: the chosen path, estimated output, gas, net score, and the algorithm label that produced each candidate.

What is verifiable

CoW Explorer verifies auction outcomes. Etherscan verifies settlement transactions for wins. Fynd's public routing logic makes candidate evaluation inspectable — no black-box claims required.

Run it yourself

Stream live CoW orders, solve them locally with Fynd, then compare route decisions against PropellerSwap's public feed. Solve-only; submitting settlements requires being a registered CoW solver.

Local loop

Start Fynd locally, subscribe to live CoW orders, run the same route search, and compare candidate routes, scores, and selected paths against the production feed.

Experiment surface

Change algorithm weights, route depth, pool filters, gas assumptions, or worker-pool strategy. The point is not to trust the dashboard — it's to see whether your machine finds a better route.
